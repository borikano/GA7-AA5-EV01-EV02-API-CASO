# Separación de evidencia EV01 / EV02

Este repositorio corresponde únicamente a las evidencias:

- GA7-220501096-AA5-EV01 - Diseño y Desarrollo de servicios web - caso
- GA7-220501096-AA5-EV02 - API

No corresponde al proyecto formativo MLBT ni a las evidencias:

- GA7-220501096-AA5-EV03 - Diseño y Desarrollo de servicios web - proyecto
- GA7-220501096-AA5-EV04 - API del Proyecto

## Alcance de EV01

Para EV01 se construye un servicio web independiente que permite:

- Registrar un usuario.
- Iniciar sesión con usuario y contraseña.
- Devolver mensaje de autenticación satisfactoria si las credenciales son correctas.
- Devolver error de autenticación si las credenciales son incorrectas.
- Mantener comentarios en el código.
- Usar herramientas de versionamiento.

## Alcance de EV02

Para EV02 se realizan pruebas de la API construida en EV01 usando Postman.

Se documentan:

- Endpoints de la API.
- Cuerpos JSON de solicitud.
- Respuestas esperadas.
- Evidencias mediante pantallazos.
- Guion de apoyo para video de testing.
