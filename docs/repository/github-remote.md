## Reglas

- Crear el repositorio vacío en GitHub.
- No crear README desde GitHub si ya existe localmente.
- No crear .gitignore desde GitHub si ya existe localmente.
- No crear licencia desde GitHub si ya existe localmente.
- Conectar remoto origin después de crear el repositorio.
- Ejecutar push inicial a main.

## Comandos sugeridos

git remote add origin https://github.com/USUARIO/NOMBRE_REPO.git
git push -u origin main
