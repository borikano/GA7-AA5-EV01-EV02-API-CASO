# Evidencias offline - EV02 API

Esta carpeta contiene resultados de pruebas locales realizadas desde terminal antes de pasar a Postman.

Estas pruebas validan:

- Ruta raíz de la API.
- Endpoint de salud.
- Registro de usuarios ficticios.
- Inicio de sesión correcto.
- Consulta de perfil con token JWT.
- Error de autenticación con contraseña incorrecta.
- Error por consulta de perfil sin token.

Estas evidencias sirven como verificación técnica previa a las pruebas formales con Postman.
